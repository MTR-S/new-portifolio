Secret Word

Aplicação feita utilizando ReactJs, Javascript, HTML, CSS. O Secret Word é um jogo onde você tem como objetivo acertar a palavra secreta errando o menos possível para acumular mais pontos e evitar de perder suas vidas. Neste projeto desenvolvi habilidades de manipulação de estados, lógica de programação e estilização de componentes com o ReactJS.
